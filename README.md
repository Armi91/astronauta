
# Astronauta - Energylandia

Wear an astronaut. Game for kids in Energylandia amusement park.


## Getting Started

*Clone this repository locally:*

``` bash
git clone https://gitlab.com/dawarm/astronauta.git
```

*Install dependencies with npm:*

``` bash
npm install
```


*Run in dev mode:*
``` bash
npm run start
```

*Build:*
``` bash
npm run dist
```
